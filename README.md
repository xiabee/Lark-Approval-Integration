# Lark-Approval-Integration
Integration of Lark (AKA Feishu) approval and AWS Lambda



## 环境变量设置

| Key           | **Value**                                           |
| ------------- | --------------------------------------------------- |
| APPROVAL_CODE | 飞书审批 ID                                         |
| APP_ID        | 飞书应用 ID                                         |
| APP_SECRET    | 飞书应用密钥                                        |
| WEBHOOK_KEY   | 飞书群机器人 webhook 最后的哈希串（用于发送群消息） |
